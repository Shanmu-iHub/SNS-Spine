# SPINE: The Founder's Club

**Redesigning Common Minds & Business through SPINE**

SPINE: The Founder's Club at SNS Institutions is a networking-powered wellbeing arena where students and industry immerse in sport, fitness, gaming, entertainment, fashion, food, and coworking experiences.

## Overview

SPINE acts as a hub for networking, wellbeing, and creative hustle, fostering design thinking and premium experiences for students and industry professionals.

## Features

- **Networking:** Connect with industry leaders and like-minded peers.
- **Wellbeing:** Access to top-tier sports facilities and fitness hubs.
- **Creative Hustle:** Engage in a gaming lounge, entertainment studio, and modern coworking spaces.
- **Design Thinking:** Grounded in a mindset that promotes innovative problem-solving.

## Getting Started

1. Clone the repository.
2. Open `index.html` in a modern web browser to view the landing page.
3. No build steps are required as it utilizes plain HTML, CSS (Bootstrap 5.3.2), and JavaScript.

## Technologies Used

- HTML5
- CSS3 (Custom Styles with modern variables and animations)
- Bootstrap 5.3.2
- FontAwesome 6.5.1
- Google Fonts (Inter, Space Grotesk)
